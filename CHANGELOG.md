# Change Log

All notable changes to the "Plinko Simulator" extension will be documented in this file.

## [1.0.0] - 2025-11-16

### Added
- Initial release of Plinko Simulator
- Interactive Plinko game with physics-based ball simulation
- Multiple game configurations:
  - Row options: 8, 12, or 16 rows
  - Risk levels: Low, Medium, High
- Manual and auto-bet modes
- Virtual balance system with betting mechanics
- Sound effects for collisions and wins
- Responsive canvas-based UI with color-coded multiplier slots
- Real-time statistics and game controls
- Visual feedback for wins and game state
